Daniel's Media Explorer - API Docs

Search Images & Videos

Endpoint to search media stored in your Gun database:

GET /api/search?query=KEYWORD&type=image|video&limit=24

Example:

GET /api/search?query=nature&type=image&limit=24

Feed

Get latest media stored in your Gun database:

GET /api/feed?query=KEYWORD&limit=60

How It Works

This app fetches media from Pixabay, stores it in Gun (your personal backend), and lets users search, like, and download media directly from your app.

Notes

  • All media stored in Gun for personal API usage.
  • Users access your API, not Pixabay directly.
  • You can customize further with likes, downloads, and tags.
